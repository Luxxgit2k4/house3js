# Garden House Portfolio

Scroll-driven Three.js portfolio experience for Lakshmanan Palani.

## Features

- Animated flower garden with wind-reactive grass and flowers
- House exterior with windmill, sky, birds, and atmospheric lighting
- Scroll-based entrance, staircase climb, corridor reveal, dancing room, and terrace finale
- Free-look camera response on pointer and touch input
- Layered Web Audio ambience for garden, interior, TV lounge, stairs, and dance room

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
